# ZNAS — Portfolio Website

Premium portfolio site for [znas.io](https://znas.io). Built with Next.js, Tailwind CSS v4, GSAP + ScrollTrigger, and Lenis smooth scrolling.

## Tech Stack

- **Next.js** (App Router, TypeScript)
- **Tailwind CSS v4** with `@tailwindcss/postcss`
- **GSAP + ScrollTrigger** — scroll-driven animations
- **Lenis** — smooth scrolling

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
